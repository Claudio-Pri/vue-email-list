SNACK 5
Creare un array di oggetti in modo tale da memorizzare lo storico di temperatura di una particolare città. Ogni oggetto rappresenta uno specifico giorno, e si desidera memorizzare la temperatura minima e massima rilevata nel dato giorno.
Stampare a schermo il giorno più freddo ed il giorno più caldo.
